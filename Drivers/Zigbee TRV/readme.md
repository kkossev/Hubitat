## Thermostats and TRVs drivers for Hubitat Elevation 

This folder contains several different drivers supporting different thermostats and TRVs.
The *.groovy files need several library files, and are mostly used by the developer.
The *_lib_included.groovy files include all the neccesery libraries code and are the ones that are recommended to be used by HE users.
